# 🤖 SHO - AIエンジニア ポートフォリオサイト

AIエンジニア「SHO」のポートフォリオサイトです。
洗練されたデザインとスムーズなアニメーションで、プロフェッショナルな印象を与えます。

## ✨ 主な特徴

### 🎨 洗練されたデザイン
- **プロフェッショナルなUI**: 明るく見やすいカラースキーム（ブルー、パープル系）
- **シンプルで洗練されたデザイン**: 情報が見やすく、ごちゃごちゃしない設計
- **レスポンシブ対応**: PC、タブレット、スマートフォンで最適表示

### 🎬 スムーズなアニメーション
- **スクロールアニメーション**: 要素がふわっと表示される心地よい動き
- **タイピングエフェクト**: プロフェッショナルなキャッチコピーが自動で切り替わる
- **パーティクル背景**: 控えめで上品な動的背景
- **カウンターアニメーション**: 実績数値が自動でカウントアップ

### 💼 AIエンジニア向け機能
- **スキル表示**: AI活用、業務自動化などの専門スキルを視覚的に表示
- **学習フォーカス**: Python、GitHubなど現在学習中の技術を明示
- **制作実績**: LP制作実績を見やすく掲載
- **お問い合わせフォーム**: クライアントが簡単に連絡できる

## 🚀 使い方

### 1. ファイルの配置
すべてのファイルを同じフォルダに配置してください。

### 2. ブラウザで開く
`index.html`をダブルクリックするか、ブラウザにドラッグ&ドロップして開きます。

### 3. カスタマイズ
以下の部分をあなたの情報に変更してください。

## ✏️ 必須カスタマイズ項目

### 1. 🔗 LPのURLを設定（最重要）

**LP画像をクリックすると実際のLPに飛べるようになっています！**

#### 設定方法

`index.html`を開いて、以下の2箇所を探して編集してください：

**税理士法人のLP:**
```html
<!-- ⚠️ 重要: 下の行のURLを実際のLPのURLに変更してください -->
<a href="https://example.com/tax-lp/" target="_blank">
```
→ `https://example.com/tax-lp/`を実際のURLに変更

**ヘルスケア関連のLP:**
```html
<!-- ⚠️ 重要: 下の行のURLを実際のLPのURLに変更してください -->
<a href="https://example.com/health-lp/" target="_blank">
```
→ `https://example.com/health-lp/`を実際のURLに変更

これで、画像をクリックすると新しいタブでLPが開きます！

### 2. ⚠️ メールアドレスの変更（重要）

`index.html`の以下の部分を編集してください：

```html
<!-- ⚠️ 重要: 下の行を編集して、実際のメールアドレスに変更してください -->
<span class="detail-value" id="email-address">your-email@example.com</span>
```

→ `your-email@example.com`を実際のメールアドレスに変更

### 3. 📸 プロフィール写真の追加

#### ステップ1: 写真を準備
- 正方形の写真を用意（推奨サイズ: 500×500px以上）
- ファイル名を`profile.jpg`などわかりやすい名前に

#### ステップ2: 写真をフォルダに配置
ポートフォリオフォルダに写真を配置します。

#### ステップ3: HTMLを編集
`index.html`の以下の部分を編集：

```html
<!-- 現在（コメントアウトされている） -->
<!-- <img src="your-photo.jpg" alt="SHOのプロフィール写真" class="profile-photo"> -->

<!-- 以下のように変更（コメントを外す） -->
<img src="profile.jpg" alt="SHOのプロフィール写真" class="profile-photo">
```

そして、プレースホルダーをコメントアウト：
```html
<!-- プレースホルダーは不要になるのでコメントアウト -->
<!-- 
<div class="avatar-placeholder">
    <i class="fas fa-user-astronaut"></i>
    <div class="photo-instruction">写真をここに配置</div>
</div>
-->
```

### 4. 制作実績の画像追加（推奨）

#### 税理士法人のLP画像
`<div class="project-placeholder">`の部分を探して、以下のように変更：

```html
<!-- 画像を追加する場合 -->
<img src="tax-lp.jpg" alt="税理士法人LP" style="width:100%; height:100%; object-fit:cover;">
```

#### ヘルスケア関連のLP画像
同様に変更：

```html
<img src="health-lp.jpg" alt="ヘルスケアLP" style="width:100%; height:100%; object-fit:cover;">
```

## 🔧 技術スタック

### 使用技術
- **HTML5**: セマンティックなマークアップ
- **CSS3**: Flexbox、Grid、カスタムプロパティ、アニメーション
- **Vanilla JavaScript**: Intersection Observer API、Canvas API
- **Google Fonts**: Orbitron、Rajdhani
- **Font Awesome 6**: アイコンライブラリ

### 主要機能の仕組み

#### スクロールアニメーション
Intersection Observer APIを使用して、要素が画面に入ったタイミングでアニメーションを発火

#### パーティクル背景
Canvas APIを使用して動的なパーティクルを描画
- パーティクル数: 50個（軽量化）
- 接続距離: 120px
- 控えめな透明度で上品な演出

#### タイピングエフェクト
文字を1文字ずつ表示・削除して、複数のキャッチコピーを自動で切り替え
